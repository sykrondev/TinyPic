# TinyPic

TinyPic is the lightest portable screenshot companion for Windows: fast region captures, instant clipboard flow, and a playful pinkcore interface that stays out of your way.

![TinyPic settings](assets/tinypic-settings.png) <img width="538" height="559" alt="screenshot_2026-05-20_04-04-12" src="https://github.com/user-attachments/assets/2801e1d8-b4c9-4b70-82f6-062df75c6524" />


TinyPic is a small Windows screenshot tool with a pinkcore frameless interface, animated falling stars, region capture, preview, clipboard support, hotkeys, and optional Windows startup.

## About

TinyPic is built for quick screenshots without heavy menus, installers, or background clutter. It opens minimized, stays portable, and gives you a simple capture workflow for grabbing regions, previewing images, copying them to the clipboard, and switching capture modes from the tray.

The app keeps its personality with a frameless pinkcore UI and animated falling stars, while staying focused on being a lightweight utility you can launch and use immediately.

## Usage

it's start minimized so, click on it for region photo

right click for settings and other picture modes.

## Features

- Portable Windows `.exe`
- Region screenshot capture
- Image preview before saving or copying
- Clipboard support
- Hotkeys
- Optional Windows startup
- Small pinkcore frameless interface

## Config

User settings are stored in:

```text
%APPDATA%\TinyPic\config.json
```
